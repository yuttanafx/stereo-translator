# 🎧 Stereo Voice Translator

แปลภาษาด้วยเสียง แยกหูซ้าย-ขวาจริง ผ่าน Bluetooth Headphone

## Deploy บน Vercel (วิธีง่ายที่สุด)

### วิธีที่ 1: Drag & Drop (ไม่ต้อง GitHub)
1. ไปที่ https://vercel.com/new
2. ลาก folder `stereo-translator` ทั้ง folder ไปวาง
3. กด Deploy — เสร็จใน 1 นาที!

### วิธีที่ 2: ผ่าน GitHub
1. สร้าง repo ใหม่บน GitHub
2. อัปโหลดไฟล์ทั้งหมดเข้า repo
3. ไปที่ vercel.com/new → Import GitHub repo → Deploy

## ใช้งาน
1. ใส่ **Gemini API Key** (ขอฟรีที่ aistudio.google.com/app/apikey)
2. ตั้งภาษาของคน A และคน B + เลือกเสียงแต่ละฝั่ง (เสียงดึงจากระบบ/เบราว์เซอร์อัตโนมัติ)
3. สนทนาแบบสลับกันพูด:
   - คนที่จะพูด **กดปุ่มของตัวเอง (A หรือ B) ก่อนพูด** — ต้องกดเพื่อให้ระบบรู้ว่าจะฟังภาษาไหน
   - พูดจบ ระบบ**แปลและพูดออกเสียงคำแปลให้อีกฝ่ายฟังทันที** โดยอัตโนมัติ ไม่ต้องกดอะไรเพิ่ม
   - เสียงแปลจะออกหูฝั่งภาษาปลายทาง (A พูด → แปลออกหู R ฝั่ง B, B พูด → แปลออกหู L ฝั่ง A)
   - ไม่พูดเสียงต้นฉบับซ้ำ พูดออกเฉพาะคำแปลเท่านั้น
   - พูดจบรอบหนึ่งแล้วกดปุ่มของอีกฝ่ายเพื่อพูดต่อ สลับกันไปเรื่อยๆ เหมือนคุยกันจริง
   - มีประวัติการสนทนาแสดงด้านล่างให้ดูย้อนหลังได้

## Tech Stack
- **Next.js 14** (App Router)
- **Web Speech API — SpeechRecognition** — รับเสียง (STT) เลือกภาษาฟังตามปุ่ม A/B ที่กด
- **Google Gemini 2.5 Flash** (ผ่าน AI Studio API key) — แปลภาษา ฟรีและเร็ว
  - หมายเหตุ: `gemini-2.0-flash` ถูก Google ปลดระวางแล้ว (ปิดให้บริการ 1 มิ.ย. 2026)
    ถ้าเจอ error 429/404 ในอนาคต ให้เช็ครายชื่อโมเดลล่าสุดที่ ai.google.dev/gemini-api/docs/models
- **Web Speech API — SpeechSynthesis** — อ่านออกเสียงคำแปล (TTS) ฟรี ทำงานในเบราว์เซอร์ ไม่ต้องมี backend
- เสียงแปลออกหูตามภาษาปลายทาง: ◀ หูซ้าย = ภาษา A, หูขวา ▶ = ภาษา B
  - หมายเหตุ: Web Speech API ไม่ส่งคืน audio buffer ให้ใช้ AudioContext StereoPanner ได้ตรงๆ
    เหมือน MP3 ของ TTS API ทั่วไป ดังนั้นเสียงแปลแต่ละรอบจะออกแค่หูเดียวตามฝั่งปลายทาง
    ไม่ใช่การ pan สัญญาณเดียวพร้อมกันสองหูจริง

## ทำไมต้องกดปุ่มก่อนพูด?
Web Speech API ต้องรู้ภาษาที่จะฟังล่วงหน้า (ไม่มี auto-detect ภาษาในตัว) การกดปุ่ม A/B
ก่อนพูดทำให้ระบบแปลถูกทิศทางแม่นยำ 100% ถ้าใช้หูฟัง Bluetooth ที่มีปุ่มกด สามารถ map
ปุ่มบนหูฟังให้กดแทนปุ่มในแอปได้เช่นกัน
